this指向

本文最后更新于：2020-08-27 22:13:46

this的值取决于运行时上下文，非定义时上下文。

由于函数在内存中独占一块空间，当对象的属性为函数时，实际将函数地址赋值给属性的value值。

this的使用场景主要有4种。call(),apply(),bind()函数赋值，构造函数调用，普通函数调用，对象函数调用。

对象函数内存图.jpg

call/apply/bind

显式绑定

call()方法使用一个指定的this及一个或多个参数来调用函数。function.call(thisArg, arg1, arg2, ...)，当第一个参数为null或undefined，this在浏览器环境下指向window对象。

apply()方法使用一个指定的this及一个数组或类数组对象参数。func.apply(thisArg, [argsArray])。

bind()方法会创建一个新函数，当bind()被调用时，第一个参数作为指定的this，其余参数作为新函数参数，在调用时使用。因此，bind()能够使函数具有预设参数。function.bind(thisArg, arg1, arg2, ...)。

call()与apply()的区别在于call()方法接受的是参数列表，apply()方法接受的是参数数组。

let obj = {
  sum: 0,
  arr: [1, 2, 4],
};
/** apply **/
let max = Math.max.apply(obj.arr, [8, 88]);
console.log(max); // 88
/** call **/
function foo(a, b) {
  console.log(a + b);
}
foo.call(obj.sum, 3, 9); // 12
/** bind **/
let fn = foo.bind(null, 20);
fn(45); // 65

构造函数

new绑定

构造函数中this指向构造函数使用new关键字产生的实例对象，无法改变this指向。

new关键字主要步骤：

  1. 创建一个空对象foo = {}
  2. 设置对象的_proto_，foo._proto_ = Foo.prototype
  3. 将构造函数的this指向该新对象foo
  4. 执行构造函数中的逻辑代码
  5. 如果构造函数中没有return语句，返回对象foo的地址；如果构造函数中含有return语句且返回对象类型new_foo，则返回该对象类型new_foo地址，非foo地址；如果构造函数中含有return语句且返回类型为值(基本)类型，则返回foo的地址

普通函数

默认绑定

普通函数调用，this指向全局对象，浏览器环境为window，node环境为global，方式：函数名(...)

/** 情境一 **/
function foo() {
  var a = 2;
  console.log(this.a); // 3
}
var a = 3;
foo(); // foo() === window.foo()，此时this指向全局变量，全局变量的a = 3，因此打印3

/** 情境二 **/
let obj = {
  b: 1,
  foo() {
    console.log(this.b);
  }
}
let temp = obj.foo;
temp(); // undefined,	此时this指向全局对象，全局对象中不含b属性，因此打印undefined

对象函数

即函数作为对象方法被调用，此时this指向该对象，方式：对象名.函数名(...)。

隐式绑定

let obj = {
  name: "obj",
  fn() {
    console.log(this.name);
  },
};
let name = "window";
obj.fn(); // 'obj' obj.fn() === obj.fn.call(obj)

回调函数

回调函数中的this默认指向全局对象，即函数参数传入callback，并在函数内调用callback()，此时为函数名(...)。

/** 情境一 **/
let obj = {
  num: 1,
  foo() {
    function fn() {
      console.log(this.name);
    }
    fn();
  },
};
obj.foo(); // this指向window

/** 情境2 **/
let obj = {
  num: 1,
  foo(callback) {
    callback();
  },
};
obj.foo(function fn() {
  console.log(this.name); // this指向window
});

箭头函数

箭头函数并且没有自己的this，arguments，super或new.target，由外围最近一层非箭头函数的this决定。

箭头函数不能作为构造函数，主要原因在于，第一，箭头函数没有自己的this，无法调用call/apply；第二，没有prototype属性，而new 命令执行时需要将构造函数的prototype赋值给对象的_proto_。
