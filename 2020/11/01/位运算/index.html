位运算

本文最后更新于：2020-11-08 17:30:06

符号 含义
｜ 按位或
& 按位与
^ 按位异或
～ 按位取反
>> 左移
<< 右移

判断奇偶

根据最末位是0还是1来决定，0则为偶数，1则为奇数。

if (a & 1) {
  console.log('a 为奇数')
} else {
  console.log('a 为偶数')
}

交换数值

实现不借助第三方变量的方式交换两个数

a ^= b
b ^= a
a ^= b

变换正负

正数的二进制表示为原码，负数的二进制表示为补码。对二进制数取反+ 1即可变换符号

b = ~a + 1

获取绝对值

任何数和0异或不变，和-1异或相当于取反。若sign为-1则- sign即为+ 1

let sign = a >> 31 //正数补0，sign=0；负数补1，sign=-1
return sign >= 0 ? a : ~a + 1 // 第一种
return (a ^ sign) - sign // 第二种

二进制中1的个数

while (n > 0) {
  count++
  n &= n - 1
}

高低位交换

b = (a << 8) | (a >> 8)

概念速记

使用x & 1 == 1 判断奇偶数

两个相同的数异或是0，一个数和0异或的结果是它本身

使用x ^ y >= 0判断符号是否相同。两个正数或两个负数异或结果为0，一正一负结果为1

n & (n - 1)作用为将n的最右侧的1置为0，因此循环操作，即可获取n的1的个数。n & (n - 1) = 0,则N为2的幂值
