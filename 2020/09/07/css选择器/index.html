CSS选择器

通配选择器

*，匹配所有html元素，性能差，不建议使用。

标签选择器

匹配html元素标签。

类选择器

匹配html元素class属性。

ID选择器

匹配html元素id属性。

* {
  margin: 0;
  padding: 0;
} 
div {
  width: 100px;
  height: 100px;
}
.container {
  font-size: 20px;
  background-color: #0f0;
}
#footer {
  line-height: 1.5em;
  text-decoration: underline;
}

组合选择器

利用组合选择符对选择器进行排列组合。

  • 后代【子元素、孙子元素等】组合器：以空格分隔。
  • 直接子代组合器：以大于号分隔。
  • 紧邻兄弟组合器：以加号分隔。加号两边为兄弟元素，拥有相同父元素，后一个元素紧跟前一个元素之后，非兄弟元素样式设置将无效。
  • 一般兄弟组合器：以波浪号分隔，表示左边元素之后的所有相邻兄弟元素的样式设置。
div p {
  text-indent: 2em;
  text-transform: uppercase;
}
.root > p {
  border: 1px dashed #00f;
  color: #f00;
}
#btn + p {
  text-shadow: 2px 2px 1px #000;
  font-weight: lighter;
}
li ~ li {
  font-family: Arial, Helvetica, sans-serif;
  float: right;
}

伪类选择器

表示被选择html元素的特殊状态。

:link：匹配未被访问的链接标签。

:visited：匹配已经被访问的链接标签。

:hover：鼠标悬停效果。

:active：鼠标按下到抬起的过程的选中状态。

a:link {
  background-color: #eee;
  font-weight: bolder;
}
a:visited {
  background-color: #0f0;
  font-weight: normal;
}
a:hover {
  text-decoration: none;
  background: transparent;
}
a:active, button:active {
  color: #f00;
  border: 10px solid #f00;
}

根据LVHA规律，往往以:link-:visited-:hover-:active的顺序进行css排布

(E):first-child：兄弟元素中第一个元素为E【css4所选元素的父元素不必须】。

(E):first-of-type：兄弟元素中第一个E元素【css4所选元素的父元素不必须】。

(E):last-child：兄弟元素中最后一个元素为E。

(E):last-of-type：兄弟元素中最后一个E元素。

(E):nth-child(an+b), n = (0, 1, 2, ...) ：正向，以所有兄弟元素为基准，匹配an+b表达式且元素为E。

(E):nth-last-child(an+b), n = (0, 1, 2, ...)：反向，以所有兄弟元素为基准，反向匹配an+b表达式且元素为E。

(E):nth-of-type(an+b), n = (0, 1, 2, ...)：正向，以兄弟元素中所有E元素为基准，匹配an+b表达式。

(E):nth-last-of-type(an+b), n = (0, 1, 2, ...)：反向，以兄弟元素中所有E元素为基准，反向匹配an+b表达式。

p:first-child {
  background: linear-gradient(to right, #f00, #0f0);
  opacity: 0.5;
}
a:first-of-type {
  overflow: hidden;
  position: absolute;
}

属性选择器

根据属性名或属性值进行匹配。

[attr]：匹配属性名attr。

[attr=value]：匹配属性名为attr，属性值为value的元素。

[attr~=value]：匹配属性名为attr，属性值含有value，value值不可拆分字符串，'value' => value。

[attr*=value]：匹配属性名为attr，属性值含有value，value值可拆分，'vvvvvalueeee' => value。

[attr|=value]：匹配属性名为attr，属性值开头为value或value-，value值不可拆分字符串。

[attr^=value]：匹配属性名为attr，属性值开头为value，value值可拆分。

[attr$=value]：匹配属性名为attr，属性值结尾为value，value值可拆分。

[attr=value i]：匹配属性名为attr，i或I会忽略属性值的大小写。

分组选择器

以逗号分隔，表示多个元素设置同一样式，一般用于设置公共样式。

p, div {
  color: #000;
  position: fixed;
}

伪元素

允许对被选择元素的特定部分进行修改。

::before, ::after

::first-letter选中块级元素首行第一个字母或汉字。

::first-line选中块级元素首行。

::selection选中高亮。

div::before {
  content: '《';
}
div::after {
  content: '》';
}
p::first-letter {
  color: #000;
  font-size: 30px;
}
p::first-line {
  background-color: #0f0;
  line-height: 10px;
}
p::selection {
  color: #0f0;
  background-color: #00f;
}
